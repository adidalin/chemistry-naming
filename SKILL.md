## 命名方案（供用户选择）

### 方案A：简洁数字式（排序友好）

**适用场景**：专题资料、知识清单、快速浏览

```
文件夹：1单元_走进化学世界/
课时： 1.1物质的变化和性质/
文件：  1.1物质的变化和性质_教案.doc
复习：  第一单元_复习课_教案.docx
检测：  第一单元_单元检测.docx
答案：  第一单元_答案.docx
综合：  第一单元_主题综合实践作业.docx
```

### 方案B：完整数字式（正式规范）

**适用场景**：正式备课目录、教学资源库

```
文件夹：01_第一单元/
课时：  1.1物质的变化和性质/
文件：  1.1物质的变化和性质_教案.doc
单元备课：第一单元_单元备课.doc
```

### 方案C：课题序号式（按教材课题编号）

**适用场景**：严格按教材课题划分

```
课时文件夹：课题1_物质的变化和性质/
文件：       课题1_物质的变化和性质_教案.doc
子课时：     课题2.1_常见的酸_教案.docx
实验活动：   实验活动1_氧气的实验室制取与性质_教案.docx
跨学科：     跨学科1_微型空气质量检测站的组装与使用_教案.docx
```

### 方案D：纯知识清单式

**适用场景**：知识梳理、母题整理

```
文件夹：1单元_走进化学世界/
文件：  第一单元_知识清单及母题.docx
       第一单元_答案.docx
```

## 命名规则速查

### 规则1：各部分用下划线 `_` 分隔，不用空格

### 规则2：课时编号格式

| 层级 | 格式 | 示例 |
|------|------|------|
| 主课题 | `{单元号}.{课题号}{课题名}` | `1.1物质的变化和性质` |
| 子课题 | `{单元号}.{课题号}.{子序号}{子课题名}` | `1.2.1化学实验与科学探究` |
| 拓展主题 | `{课题号}{主课题}_{拓展主题}` | `8.2高铁建设中金属材料的获取` |
| 实验活动 | `实验活动{编号}` | `实验活动1` |
| 跨学科实践 | `跨学科实践{编号}` | `跨学科实践1` |
| 绪论 | `绪论` | `绪论_教案.doc` |

### 规则3：文件类型统一用词

- 统一用 `学历案`，不用 `学案`
- 统一用 `单元检测`，不用 `单元测试` / `检测卷` / `测试卷`
- 统一用 `答案`，不用 `题库答案` / `母题答案` / `测试卷答案`
- 统一用 `知识清单及母题`，不用 `知识清单+母题` / `知识清单+题库`

### 规则4：资源视频命名

```
{单元编号}_{课时编号}_{描述性名称}.mp4
```

示例：`08_8.1_导入.mp4`、`09_实验活动_媒体1.mp4`、`10_氯化钠.mp4`

## 工作流程

当用户提出整理/重命名化学文件时，按以下流程操作：

### 步骤1：分析当前结构

读取目标目录，列出所有文件和文件夹，了解当前命名状况。

### 步骤2：识别问题类型

- 名称不一致（如 `学案` vs `学历案` 混用）
- 格式不规范（空格、错字、冗余）
- 结构混乱（文件放错位置、多个文件夹混合）
- 缺少分类（视频文件无意义命名）

### 步骤3：提供方案供选择

向用户展示上述命名方案（A/B/C/D），询问偏好。如果用户有特殊要求，根据规则定制。

### 步骤4：执行

- 创建新文件夹（如有需要）
- 移动文件到正确位置
- 重命名文件和文件夹
- 标注废弃文件夹为 `[废]xxx` 等待用户确认删除

### 步骤5：验证

列出最终结构，确认所有文件已正确命名。

---

## 注意事项

1. **不删除文件**：只重命名、移动，不执行删除操作
2. **废弃文件夹标注**：用 `[废]` 前缀标记，让用户自行确认删除
3. **保留扩展名**：不改动文件格式（不转换 `.doc` ↔ `.docx`）
4. **备份意识**：大量操作前建议用户先备份
